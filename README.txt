Веб-приложение File Manager представляет собой удобный инструмент для организации и управления файлами в персональном облачном хранилище.
Основная задача приложения - предоставить пользователю интуитивно понятный интерфейс для выполнения базовых операций с файлами и папками.
Техническая реализация проекта основана на микрофреймворке Flask, что обеспечивает легкость развертывания и минимальные требования к системным ресурсам.
Для хранения метаданных пользователей используется SQLite(sqlalchemy).
Интерфейсная часть построена на чистом HTML/CSS с минимальным использованием JavaScript для обеспечения интерактивности.
Ключевыми преимуществами решения являются его простота развертывания, низкие системные требования и понятный интерфейс.
